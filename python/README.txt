Info:
#----------------------------------------------------------------------------#

This project contains a number of python scripts to run the C++ modules 
from the meanie3D software package. The scripts are designed to take a
lot of the complexity of running the 2D/3D clustering and tracking code
through a simple JSON configuration file. 

The tasks performed in this fashion by the python code include:
- clustering
- tracking
- visualisation of source data, clusters and tracks
- tracking statistics 
- cross-scale comparisons

In order to find out more about the meanie3D project, please visit the
following site: http://git.meteo.uni-bonn.de/projects/meanie3d

Required for all systems:
Python 2.7 and higher ( http://python.org )
setuptools 0.6 or higher (http://peak.telecommunity.com/DevCenter/setuptools)

#----------------------------------------------------------------------------#
INSTALLATION:
#----------------------------------------------------------------------------#

This package is installed as part of the meanie3D software installation.
